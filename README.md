This sample project to test open world on mobile

API:OpenGL ES3.1 and Metal  
Target HW: Galaxy S8(Snap835), iPhone 6s plus(A9)  
Target Memory buget: 600mb  
iPhone6s: 837MB/1396MB/2048MB/68% (iOS 9.2)
